## kubecm switch

交互式切换 Kube Context

### 简介


交互式切换 Kube Context

**可以通过键入 `\` 来进行搜索**

```
kubecm switch [flags]
```

![switch](../../static/switch.gif)

### 示例

```

# Switch Kube Context interactively
kubecm switch

```

### 选项

```
  -h, --help   help for switch
```

### 全局选项

```
      --config string   path of kubeconfig (default "/Users/guoxudong/.kube/config")
```
