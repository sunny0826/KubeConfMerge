## kubecm create（实验性）

生成 KubeConfig（实验性）

### 简介

交互式生成 KubeConfig（实验性）

```
kubecm create
```

### 示例

```

# Create new KubeConfig(experiment)
kubecm create

```

### 选项

```
  -h, --help   help for create
```

### 全局选项

```
      --config string   path of kubeconfig (default "/Users/saybot/.kube/config")
```
