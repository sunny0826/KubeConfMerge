## kubecm alias

Generate alias for all contexts

### Synopsis

Generate alias for all contexts

```
kubecm alias
```

### Examples

```

$ kubecm alias -o zsh
# add context to ~/.zshrc
$ kubecm alias -o bash
# add context to ~/.bash_profile

```

### Options

```
  -h, --help         help for alias
  -o, --out string   output to ~/.zshrc or ~/.bash_profile
```
