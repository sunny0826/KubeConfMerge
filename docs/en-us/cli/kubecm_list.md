## kubecm list

List KubeConfig

### Synopsis

List KubeConfig

```
kubecm list
```

### Examples

```

# List all the contexts in your KubeConfig file
kubecm list
# Aliases
kubecm ls
kubecm l
# Filter out keywords(Multi-keyword support)
kubecm ls kind k3s
# Useful environment variables
KUBECM_DISABLE_K8S_MORE_INFO: it will disable the k8s more info in the output

```
