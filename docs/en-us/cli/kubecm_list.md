## kubecm ls

List KubeConfig

### Synopsis

List KubeConfig

```
kubecm list
```

### Examples

```

# List all the contexts in your KubeConfig file
kubecm list
# Aliases
kubecm ls
kubecm l
# Filter out keywords(Multi-keyword support)
kubecm ls kind k3s

```
