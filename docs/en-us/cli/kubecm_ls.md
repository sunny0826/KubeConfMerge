## kubecm ls

List KubeConfig

### Synopsis

List KubeConfig

```
kubecm ls
```

### Examples

```

# List all the contexts in your kubeconfig file
kubecm ls
# Aliases
kubecm l
# Filter out keywords(Multi-keyword support)
kubecm ls kind k3s

```

### Options

```
  -h, --help            help for ls
```
