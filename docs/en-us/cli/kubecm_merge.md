## kubecm merge

Merge the KubeConfig files in the specified directory

### Synopsis

Merge the KubeConfig files in the specified directory

```
kubecm merge [flags]
```

### Examples

```

# Merge kubeconfig in the dir directory
kubecm merge -f dir

```

### Options

```
  -f, --folder string   Kubeconfig folder
  -h, --help            help for merge
```
